About

Project Robot Playground is a collection of simulation assets to make it easy to test your robot in realistic scenes. Each set includes highly curated models that have been optimized for simulation. Features like low-poly meshes with realistic textures, simple shape collisions, realistic inertias and masses have been added.

  • Collection of curated simulation assets.

  • Commercial license friendly.

  • Gazebo integration and support.
